## Project Overview
SIMPLE REGULATOR is a compact 2-layer linear voltage regulator PCB designed for beginner and academic use.  
This project demonstrates basic voltage regulation, PCB layout practices, and component placement using minimal components.

---

## Key Features
- Simple linear voltage regulator design  
- 2-layer PCB  
- Compact and minimal footprint  
- Input and output status LEDs  
- Beginner-friendly educational project  

---

## Components Used
| Reference | Component |
|---------|----------|
| U1 | Linear Voltage Regulator IC |
| R1, R2 | Resistors |
| C1, C2, C3 | Decoupling and filter capacitors |
| D1, D2 | LED indicators |
| J1 | Input connector |
| J2 | Output connector |

---

## Working Principle
1. DC input is supplied through connector J1  
2. Input LED (D1) indicates power availability  
3. The regulator IC (U1) stabilizes the input voltage  
4. Capacitors filter noise and improve voltage stability  
5. Regulated output is available at connector J2  
6. Output LED (D2) confirms regulated voltage

## How to Use
1. Provide DC input voltage at J1  
2. Ensure input voltage is within regulator specifications  
3. Check input LED for power indication  
4. Use regulated output from J2
